Loki Dashboard
​
CATS uses Loki as the primary log aggregation system, which provides a modern and efficient log management solution optimized for Kubernetes environments. Loki takes a lightweight approach by indexing only metadata, such as labels, making it more cost-effective and scalable, especially for teams dealing with high volumes of logs.
As there is no standalone user interface for Loki, the only way to query Loki for logs is through Grafana. Please see the
Grafana
documentation for utilizing the tool.
