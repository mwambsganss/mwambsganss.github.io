Overview

This document outlines the Model Context Protocol (MCP) servers planned for Lilly systems. These MCP servers will enable AI agents to access various Lilly tools and resources, with a particular focus on resource provisioning capabilities. The Lilly Cloud MCP server specifically could be used in the Agentic AI registry project to help provision resources.
