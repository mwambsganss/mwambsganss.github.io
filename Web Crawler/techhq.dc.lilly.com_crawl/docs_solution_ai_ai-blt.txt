Ethics & Compliance Expense Analysis
‚Äã
AADS is using Textract Expense Analysis to pull text from receipts and invoices submitted to Concur Expense Reporting. The complete stack is as follows:
Source receipt images in S3.
Lambda function to start expense analysis.
Textract sends notification to SNS Topic when job completes.
Message from SNS Topic is written to SQS.
Lambda function listens to SNS Topic. Writes completed analysis output to S3 as a JSON document.
DynamoDB is used to sync source, Textract job ID, and target.
