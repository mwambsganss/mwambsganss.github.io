VPN on Demand (MobileIron)‚Äã


PositionRetired 2020-12-31
Tagscybersecurity
Information SensitivityRed
ArchitectRyan Elkins
Contact(s)Ryan Elkins


Limited Availability service that should only be used if the application is not able to surface data through the API gateway


Notes‚Äã


VPN on Demand is a service for Lilly managed mobile devices that allows for the launch of a restricted VPN tunnel (Production Profile) when the user is accessing specified internal web applications such as LillyNet.  This VPN tunnel will trigger 'On Demand' by automatically launching when a user attempts to browse to one of the specified URLs. VPN on Demand is a Limited Availability service that should only be used if the application is not able to surface data through the API gateway.
