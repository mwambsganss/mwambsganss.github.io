Lilly MCP Servers

Last Updated: July 25, 2025

Owner: Brian Lewis

Point of Contact: Ali Kharazmi

Overviewâ€‹


This document outlines the Model Context Protocol (MCP) servers planned for Lilly systems. These MCP servers will enable AI agents to access various Lilly tools and resources, with a particular focus on resource provisioning capabilities. The Lilly Cloud MCP server specifically could be used in the Agentic AI registry project to help provision resources.


MCP Serversâ€‹


The following MCP servers are planned or under consideration:


    

  • Lilly Cloud MCP: Resource provisioning for cloud infrastructure

  • Cortex Docs MCP: Access to Cortex documentation

  • CATS MCP: Internal application deployment (built)

  • GitHub MCP: Code repository access
    • 

  • Lilly Flow MCP: Workflow automation

  • Lilly Kubed MCP: External application deployment (hanger- built)
