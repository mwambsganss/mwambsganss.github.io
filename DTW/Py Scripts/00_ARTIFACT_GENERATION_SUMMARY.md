## Key Findings from Workshop

### Top Pain Points Identified:
1. Roles constantly changing - impossible to keep up with who does what
2. Too many handoffs create loss of control and communication gaps
3. No clear standard on grid ownership, trafficking, and fixing failures
4. Black boxes - unclear who owns what step
5. 5-15 day QA process creates delays

### Top Big Ideas:
1. ⭐ Automated grid notifications when steps complete
2. Clear role clarity documentation (RACI)
3. ⭐ Metadata auto-population from source systems
4. Teams channel pings tagging next person
5. Fast-track path for high-urgency messages

### Agreed Hills (Success Outcomes):
1. **Marketers and Media Teams** complete banner trafficking from AFD to live in **1 day** with **zero communication gaps** → **80% time reduction**
2. **All stakeholders** know exactly who owns each step with **automated notifications** → **100% role clarity**
3. **Agencies** receive **complete, accurate, error-free assets** in single automated handoff → **90% reduction in QA rework**

## How to Use These Artifacts

### For Immediate Use:
1. Open any .pptx file in PowerPoint or Google Slides
2. Review content for accuracy against your needs
3. Apply Lilly brand templates if desired
4. Customize fonts to match brand guidelines (suggest: Patrick Hand or Caveat for hand-written style)

### For Design Enhancement:
If you want to enhance the visual design:
1. Apply official Lilly PowerPoint templates
2. Replace Comic Sans MS with branded hand-written fonts (Patrick Hand, Caveat)
3. Add texture overlays to sticky notes for more organic appearance
4. Enhance backgrounds with actual kraft paper textures
5. Add hand-drawn arrows and connector lines
6. Import into design tools (Figma, Adobe Illustrator) for final polish

### For PNG Export:
To convert slides to PNG images (1920px width):
1. Open each .pptx in PowerPoint
2. File → Export → Change File Type → PNG
3. Select "Every Slide" and set resolution to 1920x1080
